## Project Setup

1. Create a `src` folder with the following structure:
   src/
   ├── controllers/
   ├── database/
   ├── model/
   ├── routes/
   └── types/

2. Inside `src`, create and write the initial setup code in:
   - `index.ts`
   - `database/database.ts`

3. Define your data models:
   - Create the `Link` and `Count` models in `model/`

4. Define your custom types:
   - Create the `Link` type in `types/`

5. Write code for ShortenLinkController.
